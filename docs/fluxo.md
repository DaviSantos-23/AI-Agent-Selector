# Fluxo de Funcionamento

## Passo 1

O usuário abre a aplicação.

↓

## Passo 2

Seleciona a tarefa desejada.

- Gerar Texto
- Resumir Texto
- Chat

↓

## Passo 3

Digita o prompt.

↓

## Passo 4

O sistema identifica qual agente executar.

↓

## Passo 5

É realizada uma requisição para a API da Hugging Face.

↓

## Passo 6

O modelo processa a solicitação.

↓

## Passo 7

A resposta é retornada para a aplicação.

↓

## Passo 8

O usuário visualiza o resultado.
